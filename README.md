# Local Music Tag Web
为ncm_crack做的功能附加，能够自动解析163 Key并通过id直接精确抓取对应数据，支持普通歌曲与电台节目

可能解析失败或者访问失败，fallback到原始方案

添加了过滤已成功项目的功能，包括目录显示与批量刮削

本地构建部署
